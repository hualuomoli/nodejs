# 介绍

前端框架使用，使用 jquery bootstrap font-awesome angular angular-ui-router

# 规范
所有代码需要符合angular的[规范](https://github.com/johnpapa/angular-styleguide/blob/vscode/snips/i18n/zh-CN.md)

# 目录结构

> + app.js                  应用主入口
> + core                    核心模块
> + blocks                  通用模块
>     * logger                   日志模块
>     * exception                异常处理模块
>     * routehelper              路由辅助模块



# 如何使用

下载项目，执行`npm install & bower install & gulp`
